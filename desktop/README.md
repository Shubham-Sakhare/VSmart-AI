## 🚀 About

VSmart AI is an AI-powered desktop assistant inspired by Jarvis.

Built with:

- Electron
- React
- TypeScript
- Vite
- Node.js
- AI Models
- Voice Recognition
- System Monitoring


## ✨ Features

✅ AI Chat Assistant  
✅ Voice Commands  
✅ System Monitoring  
✅ Application Launcher  
✅ Memory System  
✅ Desktop Automation
